# 🧪 Networking Day 2 – Static IP & Ping

## 🖥️ Topology
- PC1 → Switch (FastEthernet0/1)
- PC2 → Switch (FastEthernet0/2)
- IP Config:
  - PC1: 10.0.0.1 / 255.0.0.0
  - PC2: 10.0.0.2 / 255.0.0.0
